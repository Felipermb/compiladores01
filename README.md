# Angular 2
Para apenas executar este projeto, siga as instruções a seguir:

-> Instale este server em sua máquina (necessário o node.js instalado):
	 npm install http-server -g

-> Clone este repositório
-> Navegue até a pasta 'dist' (no terminal de comandos)
-> Execute o server com o comando:
	http-server
	
 
Angular 2 projeto usando TypeScript

Para Desenvolver:

Você precisa instalar o npm e Typescript em sua máquina e executar este projeto:
1: Clone o repositório.

2: Instale os  Pacotes
	
npm install

3: Executar Servidor

npm start
